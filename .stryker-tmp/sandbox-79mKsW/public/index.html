Case Study 3: Task Manager API

Evaluating Metamorphic Testing on a Web-Based Task Management API

This case study demonstrates Metamorphic Testing (MT) applied to a web-based task management API built with Node.js, Express, and MongoDB.

Two test suites were implemented using Playwright: a traditional baseline suite for functional correctness, and a metamorphic suite that verifies transformation-based properties such as consistency, idempotence, and reversibility.

Each API route was instrumented with manual logging to confirm controller-level coverage. Faults were injected with StrykerJS to evaluate how well each test suite detects errors.
